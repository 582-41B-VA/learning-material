# Learning material

> [!IMPORTANT]
> Always use `uv` to interact with Django:
>
> - run `uv add django` to install Django in your project, and
> - run `uv run django-admin ...` or `uv run manage.py ...` for
>   administrative tasks.
>
> Never use the Python executable directly.
